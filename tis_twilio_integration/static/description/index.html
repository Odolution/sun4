Twilio - Odoo Integration

Note: In Twilio trial account, You can send messages only to verified numbers and messages include a note about this coming from a "Twilio trial account."

This module allows to send SMS using Twilio from Odoo.

Configuration

To work this module you need to install twilio library. Use the command 'pip3 install twilio' to install the library on your system

After installing the library, Please configure the Twilio account SID, Phone number and Auth token in Settings.

You can send SMS to individual contact, multiple contacts and list of contacts. Click 'Send SMS' button to send sms. After sending SMS the state will change from Draft to Sent.
